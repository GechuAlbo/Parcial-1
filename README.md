# Portafolio Digital - Geronimo Tobio

Portafolio personal de **Geronimo Tobio**, diseñador gráfico. Un sitio web profesional que presenta trayectoria, intereses, referentes y contenido multimedia.

## 📁 Estructura del Proyecto

```
├── index.html          # Página principal - Sobre mí, trayectoria, referentes
├── multimedia.html     # Página secundaria - Video/contenido multimedia
├── styles.css          # Estilos CSS externos (todos los estilos)
├── logo.png           # Marca personal (logo)
└── README.md          # Este archivo
```

## 🚀 Publicación en GitHub Pages

### Opción 1: GitHub Pages (Recomendado)

1. **Crear un repositorio en GitHub**
   ```
   Nombre: tu-usuario.github.io
   o
   Nombre: portfolio-lab (y activar Pages después)
   ```

2. **Clonar y agregar archivos**
   ```bash
   git clone https://github.com/tu-usuario/tu-repositorio.git
   cd tu-repositorio
   
   # Copiar archivos aquí
   cp index.html multimedia.html styles.css logo.png ./
   
   git add .
   git commit -m "Agregar portafolio digital"
   git push origin main
   ```

3. **Activar GitHub Pages**
   - Ve a Settings → Pages
   - Source: main branch
   - Tu sitio estará en: `https://tu-usuario.github.io` o `https://tu-usuario.github.io/portfolio-lab`

### Opción 2: Netlify

1. **Crear cuenta en Netlify** (https://netlify.com)

2. **Drag & Drop**
   - Ir a https://app.netlify.com/sites
   - Arrastrar carpeta con los archivos
   - Automáticamente genera URL

3. **O conectar GitHub**
   - Conectar repositorio GitHub
   - Netlify despliega automáticamente

### Opción 3: Vercel

1. **Crear cuenta en Vercel** (https://vercel.com)

2. **Importar proyecto**
   - Conectar repositorio GitHub
   - Click en Deploy
   - URL automática generada
